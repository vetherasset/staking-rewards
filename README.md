# Staking Rewards

### Install

```shell
# Install nix
curl -L https://nixos.org/nix/install | sh

# Run this or login again to use Nix
. "$HOME/.nix-profile/etc/profile.d/nix.sh"

# Install dapptools
curl https://dapp.tools/install | sh
```

### Development

```shell
# Compile
dapp compile

# Test
dapp test
```
